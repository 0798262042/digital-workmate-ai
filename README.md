# 🤖 SmartDesk 

SmartDesk AI is an AI-powered workplace productivity platform designed to function as a digital co-worker. It combines intelligent tools into a single dashboard 
to help users manage emails, meetings, tasks, research, documents, and daily planning. Inspired by modern workplace technologies from contries like China,  and South Korea, 
the platform improves efficiency by automating repetitive office tasks. Key features include an AI Email Studio, Smart Planner, Meeting Hub, Research Lab, Document Analyzer, Workplace Assistant, Translator, 
Productivity Insights, Mood & Wellness Assistant, and Prompt Library. 

The application is built using HTML, CSS, JavaScript, Node.js, Firebase, and the OpenAI API. 
SmartDesk AI follows responsible AI principles by encouraging users to review AI-generated content and avoid sharing confidential information. 
Its goal is to improve productivity, simplify workplace communication, and support better decision-making through intelligent automation in the  work place
